EncryptTitan Outlook and OWA Plugin

The EncryptTitan plugin v2.0+ is designed to work on Microsoft Outlook for Windows, Outlook for OSX, and Outlook Web Access (OWA)

To load the plugin please use the Get Add-ins feature.
Then go to My Add-ins.
Next, scroll to the bottom where Custom Addins are located.
Click on the dropdown for “Add a custom add-in”
Select “Add from URL”
At the prompt, enter this link:
https://plugin.encrypttitan.net/EmailEncryption.xml
Click OK then Install.
Please ignore the warning as the plugin has not yet been verified in the Microsoft Store.

If you have any questions, please contact the EncryptTitan helpdesk via https://helpdesk.encrypttitan.com
